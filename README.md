# build
```bash
  clang++ ./Source.cpp ./PictureReader.cpp
  ./a.out ./1.bmp
  ./a.out C:\dev\1.bmp
```
Для компиляции на Windows, либо поставить clang/gcc или другой и выполнить аналогичную команду, 
либо поместить все файлы в новый "пустой проект C++" в Visual strudio.
Архив содержит весь проект.
